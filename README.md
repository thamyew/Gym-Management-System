# Gym-Management-System
This system is used to complete the day-to-day operations in a gym. For instance, tasks like registering new customer to the system, adding and managing exercise plans taught by trainers, managing the existing data for people such as staff, trainer and customer, can be done in this system. This system combines all of these functions into a single application and allows users to login into the system based on their roles to access to the features that are only available to them. Object-oriented programming concepts such as encapsulation and abstraction, as well as inheritance are also applied.
